# "Monitoring Dashboard"

Monitors and shows responses from services via http request.

## Getting Started

1. Install Node.js and npm v12.13.0

### Project setup

```bash
npm install
```

### Compiles and hot-reloads for development

```bash
npm start
```

### Compiles and minifies for production

```bash
npm run build
```
